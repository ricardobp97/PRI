Abstract:

O objetivo para este trabalho de casa era criar um Servidor Web utilizando a Framework Express que permitisse gerir os filmes disponoveis num dataset disponibilizado pelo supervisor. O dataset foi previamente carregado para uma base de dados MongoDB.

O Servidor conecta-se á base de dados através do módulo mongoose e disponibiliza uma API de dados capaz de responder a pedidos HTTP. O Servidor apresenta uma interface que mostra ao utilizador a listagem dos filmes, em que podemos filtrar a pesquisa por ano, actor ou género. São apresentados os filmes e os respetivos anos em que foram lançados. É possivel ainda entrar na página de cada filme onde são apresentados os actores e os géneros do filme.

O Servidor dá a possibilidade também de adicionar novos filmes, e por sua vez, actores e géneros ao mesmo. Dá também a possibilidade de alteração dos filmes e de remoção da base de dados, assim também como dos actores e dos géneros.

É possivel também ver a lista dos actores, ordenada por número de filmes e apresentando esse valor, de forma a mostrar um Top de actores. Tanto esta funcionalidade como os filtros de pesquisa são extras acrescentados aos objetivos para este trabalho de casa.
