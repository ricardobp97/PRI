Abstract:

O objetivo para este trabalho de casa era criar um Servidor Web para receber pedidos do tipo "/musica/X", em que o passamos em X o número da música que queremos. O servidor deveria dar como resposta o HTML correspondente á música.

Numa primeira fase foi construída uma stylesheet para transformar o arquivo de músicas em vários ficheiros XML, cada um contendo informação relativa a uma música. O nome dado aos ficheiros é da seguinte forma "docX.xml", sendo que cada música terá um número atribuído no seu nome.

De seguida foí construída uma stylesheet para transformar cada ficheiro de música em HTML.

Por fim, foi criado o Servidor em Node.js para dar resposta aos pedidos na forma especificada em cima. O servidor recebe o pedido e aplica a stylesheet á musica que corresponde ao número do pedido, e dá como resposta o HTML correspondente.
