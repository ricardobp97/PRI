Abstract:

O objetivo para este trabalho de casa era criar um Servidor Web utilizando a Framework Express que disponibilizasse uma API de Dados sobre o conteúdo das Obras.

O ficheiro que continha as obras era um ficheiro XML, pelo que foi necessário efetuar uma conversão para JSON, para que pudessemos importar o seu conteúdo para a base de dados no MongoDB.

O Servidor disponibliza a seguinte API de Dados capaz de mostrar as obras, opcionalmente, filtradas por ano, compositor e periodo, e também os compositores e as obras de um determinado compositor.

Deliverables:

  • Dataset XML
  • Dataset JSON
  • Servidor Web
